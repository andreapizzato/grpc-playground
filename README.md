# GRPC Playground
GRPC Playground is an [Electron](https://electronjs.org/) application which should help you testing and checking a GRPC service.


## Setup

```
$ yarn start
```

## Running

```
$ yarn start
```

## Build

```
$ yarn build
```

## Prettier

```
$ yarn format
```
